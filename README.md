# my-portfolio

Personal portfolio site for [Raghavendra Sai](https://github.com/raghav-dev0) — terminal-themed, built with React + Vite + TypeScript.

**Live:** https://raghav-dev0.github.io

## Stack

- React 18 + TypeScript
- Vite
- CSS Modules
- GitHub Actions → GitHub Pages

## Local dev

```bash
npm install
npm run dev
```

## Updating content

All content lives in [`src/data/resume.ts`](src/data/resume.ts). Edit that file, commit, and push — the site redeploys automatically.
